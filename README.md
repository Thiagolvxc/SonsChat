# SonsChat

Aplicación móvil de mensajería para Android desarrollada con React Native.

## Plan del proyecto

El desarrollo está dividido en **4 entregables**. Ver [PROJECT_PLAN.md](./PROJECT_PLAN.md) para el detalle completo.

| # | Entregable | Descripción |
|---|------------|-------------|
| 1 | Fundación y estructura | Proyecto Expo, navegación, pantallas base, Zustand |
| 2 | Autenticación | Firebase Auth, registro, login, sesión persistente |
| 3 | Mensajería en tiempo real | Firestore, chats, mensajes en vivo |
| 4 | Notificaciones y pulido | FCM, multimedia (imágenes/voz), optimizaciones |
